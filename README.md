## :checkered_flag: Starting ##

```bash
# Clone this project
$ git clone https://github.com/fabramosdev/react-redux

# Access
$ cd react-redux

# Install dependencies
$ yarn

# Run JSON Server First
$ yarn server

# Now, run the project
$ yarn start

# The server will initialize in the <http://localhost:3000>
```
